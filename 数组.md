# 数组

```rust
let v : [i32; 3] = [1, 2, 3];    // 数组，类型为[i32; 3]
let b : &[i32; 3] = &v;          // 数组切片（指针)，类型为&[i32; 2]，可自动转换为&[i32]
```

* 数组与数组切片是不同数据类型；
* 数组指针`&[T]`为**胖指针**，占据**两**个指针类型大小，带有数组长度信息；
* `[T; n]`是**定长**类型，`[T]`是**不定长**类型，但`&[T]`是**定长**类型，不定长类型不能作为**函数形参**和**返回值**类型，不能在栈上而是在堆上分配内存；
